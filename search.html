Search

Details on search engines

Without search engines like Google *(please don't use Bing)*, the internet would be a very difficult place to get around. We would have to memorise or keep a list of all the website URLs to type into our web address bar! /// /// Search engines circumvent this by providing the ability to look at a search term, and return a list of related websites it thinks we would want to see.

Crawling

The concept of "crawling" initially sounds weird, but makes total sense once you get it! (It doesn't stop sounding weird though). /// /// If we ourselves aren't able to keep a list of all the URLs on the internet, then how do search engines do it? In essense crawling consists of the following steps:
  1. Look at a webpage we already have
  2. Look for all of the URLs this webpage links to
  3. Follow all of these links, and add these websites to our list
  4. Repeat
It may seem a little tedious, but this is really how it works! Plus, search engines have robots to do this super fast and efficiently.

Indexing

So what do search engines do once they've "found your website"? Before we said it gets "added to their list", but in reality its a little more technical. /// /// Rather than adding to a list, a site is **indexed**, a little like pages in a book. A search engine's index can be massive, as it needs to store links to every website it can find! /// /// If a search engine doesn't have a site indexed, it means it can't show it as a potential option to the users. This is why it's important to keep track of them all.

Meta Tags

Once a search engine has found your site, it needs to read the content to decide what *the site is really all about*. This is crucial, because if a site is about horses, the engine has to know this to be able to show it to users looking for horses. /// /// Web developers (people who make websites) can give search engines a little hand, by providing a series of `meta` tags. These are tidbits of info and keywords which can help clarify what the page is offering. ```<\head> <\meta name="description" content="Learn all about HTML, CSS, and JS here!"> <\meta name="keywords" content="cool website new education web click learn html css js"> <\/head>``` They are always placed in the `head` element, and even though they can help a little, search engines have more intelligent ways of understanding pages now, so they rely on these tags less.
